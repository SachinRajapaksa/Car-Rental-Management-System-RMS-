How to compile and run.____________

Prerequisites
	Java Development Kit (JDK) : Need 8 or higher
	IntelliJ IDE : This is need to run the program

How to run 

	First open IntelliJ IDE 
	Then select the file called Vehicle RMS using Open file
	Then open the RentalApp class
	Then press Shift + F10 to run the app


Assumption We made

	Vehicle IDs are case insensitive
	Vehicle brand and models are also case insensitive
	All vehicles are available when entering to the system
	Full payment is done when renting the vehicle
